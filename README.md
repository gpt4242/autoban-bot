# 🤖 Telegram Автобан Бот — Полная версия

Бот для автобана отписчиков + лид-магниты + статистика + админ-панель.

## ✨ Функции

### 🛡️ Автобан отписчиков
- Отслеживает кто отписался от канала
- Мгновенно банит в группе/канале
- Белый список (кого не банить)
- Ручной бан/разбан

### 🎁 Лид-магниты
- Создание неограниченных магнитов по темам
- Генерация уникальных ссылок на канал
- **Проверка подписки** перед выдачей
- Отправка текста, файлов, ссылок
- Счётчик скачиваний

### 📊 Статистика
- Подписки/отписки за всё время и за день
- Источники подписок (инвайт-линки, поиск, магниты)
- Список последних подписчиков
- Ежедневные отчёты

### 🛡️ Управление банами
- Список забаненных
- Разбан по user_id
- Белый список (модераторы, свои)

### ⚙️ Админ-панель
- Inline кнопки для всего
- Логи всех действий
- Ежедневный отчёт в ЛС

## 📱 Команды

### Для админа:
| Команда | Описание |
|---------|----------|
| `/start` или `/menu` | Админ-панель |
| `/ban <user_id>` | Забанить |
| `/unban <user_id>` | Разбанить |
| `/stats` | Статистика |
| `/logs` | Последние действия |
| `/whitelist` | Белый список |
| `/addwl <user_id> [коммент]` | Добавить в вайтлист |
| `/rmwl <user_id>` | Убрать из вайтлиста |
| `/addlink <id> <ссылка>` | Добавить ссылку к магниту |

### Для пользователей:
| Команда | Описание |
|---------|----------|
| `/start` | Получить лид-магнит |

## 🎁 Как работают лид-магниты

### Создание магнита (через админ-панель):
1. Нажми **🎁 Лид-магниты** → **Создать магнит**
2. Введи тему (код): `checklist`
3. Введи название: `📋 Чек-лист успеха`
4. Отправь текст для пользователей
5. Отправь файл (PDF, документ)
6. Бот создаст **уникальную ссылку** на канал
7. Размести ссылку — люди подпишутся и получат материал

### Как пользователь получает магнит:
```
Пользователь кликает ссылку → Подписывается на канал
    ↓
Пишет /start боту → Бот проверяет подписку
    ↓
Если подписан → Отправляет магнит ✅
Если нет → Просит подписаться
```

## 🔧 Деплой на Render

Полная инструкция: `DEPLOY-RENDER.md`

Кратко:
1. `git push` на GitHub
2. Render → Web Service → подключи репо
3. Env переменные: BOT_TOKEN, CHANNEL_ID, TARGET_CHAT_ID, ADMIN_ID, PUBLIC_URL
4. Deploy!

## 📊 База данных

Бот автоматически создаёт `bot.db` с таблицами:
- `banned` — забаненные пользователи
- `whitelist` — кого не банить
- `lead_magnets` — лид-магниты и ссылки
- `subscriptions` — история подписок/отписок
- `logs` — все действия бота
- `daily_reports` — ежедневные отчёты
